# ikitravel
Ikitravel merupakan projek yang tidak tidak sengaja untuk dibuat, alias karena tidak paham dari satu matkul yang berjudul 'Pemrograman Web' saya meminta tolong kepada teman saya yang lebih jago dan paham terhadap matkul tersebut. Singkat cerita saya meminta sebuah projek kecil-kecilan yang saya harap dapat meningkatkan pemahaman saya pada matkul tersebut.

## Framework yang digunakan
- Projek ini serba vanilla alias sebisa mungkin untuk tidak memakai framework
